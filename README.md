# ☁️ Zen-Focus-Weather

O **Zen-Focus-Weather** é um script inteligente que sincroniza seu ambiente de trabalho com o mundo lá fora. 

## ✨ Funcionalidades
- Detecta automaticamente sua localização via IP.
- Consulta as condições climáticas em tempo real.
- Inicia uma playlist personalizada para aumentar sua produtividade.

## 🛠️ Tecnologias
- Python 3.10+
- Requests (para chamadas de API)
- Pygame ou VLC (para áudio)
